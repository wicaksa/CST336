What is Python?

Python is one of the most widely used programming languages in the world. It is an interpreted, object oriented, and high-level programming language with dynamic semantics used for general-purpose programming. It is an easy to learn language since it is very intuitive.

Who developed Python?

Python was developed by Guido van Rossum and was released to the public on February20, 1991. The name Python came from a very famous comedy sketch series called Monty Python's Flying Circus. Currently, it is the 3rd most powerful language in the world.

What is Python used for?

There are many applications that use Python. Some include web and internet development, scientific and numeric computing, education, desktop GUIs, software development, business appications, and games.

Why use Python for development?

Python's main goals according to the sole creator Guido van Rossum was to make an easy and intuitive language that is as powerful as the other major languages. Python is also an open source project which means that anyone can contribute to it. The code for Python is very easy to understand like plain English. Most importantly, it is suitable to be used for everyday tasks which makes development times using Python very short.
